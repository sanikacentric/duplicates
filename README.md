# 🔍 Duplicates — Find Duplicate Elements

Python solution for detecting duplicate elements in arrays/lists, a classic data structures and algorithms problem.

## 🎯 Problem

Given an array of integers, determine if any value appears at least twice. Return `True` if any duplicates exist, `False` if every element is distinct.

## 💡 Approach

- **Hash Set** — O(n) time, O(n) space using a set for constant-time lookups
- - **Sorting** — O(n log n) time, O(1) space alternative approach
